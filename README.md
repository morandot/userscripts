# UserScripts

一个存放 Tampermonkey/Greasemonkey 用户脚本的仓库。

## Scripts

| Script | Purpose | Install | Source |
| --- | --- | --- | --- |
| Text Auto Space | 为现代浏览器补充 `text-autospace` CSS 特性，改善中西文混排显示 | [GreasyFork](https://greasyfork.org/zh-CN/scripts/553348-text-auto-space) | [Source](https://github.com/morandot/userscripts/blob/main/src/text-autospace.js) |
| 微博一键拉黑 | 在微博用户名旁显示一键拉黑按钮 | [GreasyFork](https://greasyfork.org/zh-CN/scripts/553542-%E5%BE%AE%E5%8D%9A%E4%B8%80%E9%94%AE%E6%8B%89%E9%BB%91) | [Source](https://github.com/morandot/userscripts/blob/main/src/weibo-one-click-block.js) |
| CG 平台解除复制粘贴限制 | 解除指定 CG 平台网页的复制、粘贴、选中和右键限制 | [GreasyFork](https://greasyfork.org/zh-CN/scripts/574826-cg-%E5%B9%B3%E5%8F%B0%E8%A7%A3%E9%99%A4%E5%A4%8D%E5%88%B6%E7%B2%98%E8%B4%B4%E9%99%90%E5%88%B6) | [Source](https://github.com/morandot/userscripts/blob/main/src/unlock-copy-paste-on-cg-platform.js) |

## Installation

推荐使用 Tampermonkey。

1. 打开上表中的 `GreasyFork` 安装链接。
2. 在脚本管理器中确认安装。
3. 刷新目标网页后生效。
